Markdown is a lightweight markup language for creating formatted text.

## Why Markdown?

- **Simple syntax** - easy to learn and use
- **Readable** - plain text that looks good even without rendering
- **Portable** - works on any text editor
- **Widely supported** - GitHub, manyCMS, note-taking apps

## Basic Syntax

### Headings
```markdown
# Heading 1
## Heading 2
### Heading 3
```

### Emphasis
```markdown
*italic* or _italic_
**bold** or __bold__
***bold italic***
```

### Lists
```markdown
- Item 1
- Item 2
  - Nested item

1. First item
2. Second item
```

### Links and Images
```markdown
[Link text](https://example.com)
![Alt text](image.jpg)
```

### Code
```markdown
`inline code`

```python
def hello():
    print("Hello, World!")
```
```

## Extensions

Markdown supports extensions like:
- Tables
- Task lists
- Footnotes
- Definition lists
