Markdown — лёгкий язык разметки для создания отформатированного текста.

## Почему Markdown?

- **Простой синтаксис** — легко изучить и использовать
- **Читаемость** — простой текст, который хорошо выглядит даже без рендеринга
- **Переносимость** — работает в любом текстовом редакторе
- **Широко поддерживается** — GitHub, многие CMS, приложения для заметок

## Базовый синтаксис

### Заголовки
```markdown
# Заголовок 1
## Заголовок 2
### Заголовок 3
```

### Выделение
```markdown
*курсив* или _курсив_
**жирный** или __жирный__
***жирный курсив***
```

### Списки
```markdown
- Пункт 1
- Пункт 2
  - Вложенный пункт

1. Первый пункт
2. Второй пункт
```

### Ссылки и изображения
```markdown
[Текст ссылки](https://example.com)
![Альтернативный текст](image.jpg)
```

### Код
```markdown
`строчный код`

```python
def hello():
    print("Привет, Мир!")
```
```

## Расширения

Markdown поддерживает расширения:
- Таблицы
- Списки задач
- Сноски
- Списки определений

## Моё использование

Я использую Markdown каждый день для:
- Написания документации
- Создания заметок
- Написания постов в блог
- Создания README файлов
