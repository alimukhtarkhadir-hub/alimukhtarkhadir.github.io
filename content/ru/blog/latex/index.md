LaTeX — система подготовки документов, широко используемая в академической среде для научных документов.

## Почему LaTeX?

- **Профессиональное качество** — создаёт документы, готовые к публикации
- **Поддержка математики** — отличная вёрстка математических формул
- **Перекрёстные ссылки** — автоматическая нумерация рисунков и таблиц
- **Согласованность** — единообразное форматирование во всём документе

## Базовая структура документа

```latex
\documentclass{article}
\title{Мой первый документ}
\author{Ваше Имя}
\date{\today}

\begin{document}
\maketitle

\section{Введение}
Это мой первый документ LaTeX.

\section{Математика}
Вот уравнение:
\[ E = mc^2 \]
\end{document}
```

## Популярные пакеты

- `amsmath` — расширенные математические функции
- `graphicx` — включение графики
- `babel` — поддержка языков
- `natbib` — стили библиографии
